# ☕ Job Search Checklist

A soft, interactive checklist to stay accountable before sending out job applications.
Built with plain HTML, CSS, and JavaScript — no frameworks, no libraries.

## ✨ Features

- Clickable checkboxes with visual feedback
- Progress bar that updates in real time
- Warm coffee-tone color palette
- Clean, readable layout

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## 📋 What it covers

- The Basics (GitHub, Portfolio, LinkedIn, CV)
- GitHub hygiene
- Portfolio quality
- LinkedIn completeness
- CV checklist
- Daily habits

## 💡 Why I built this

To practice separating HTML, CSS, and JavaScript into clean individual files —
and to keep myself accountable during my job search as a junior frontend developer.
